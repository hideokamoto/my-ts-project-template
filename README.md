# TypeScript Package template

GitHub Template repository to create a npm package by using TypeScript.

## contribution

```bash
// clone
$ git clone git@github.com:hideokamoto/MY_REPO_NAME.git
$ cd MY_REPO_NAME

// setup
$ yarn

// Unit test
$ yarn test
or
$ yarn run test:watch

// Lint
$ yarn run lint
or
$ yarn run lint --fix

// Build
$ yarn run build

// Rebuild docs
$ yarn run doc
```
